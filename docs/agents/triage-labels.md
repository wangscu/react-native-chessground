# Triage labels

| Role | Label |
|------|-------|
| Needs evaluation | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Ready for AFK agent | `ready-for-agent` |
| Ready for human | `ready-for-human` |
| Won't fix | `wontfix` |

These are the default canonical labels. Apply them with `gh issue edit <number> --add-label <label>`.
